# Elementor FAQ

A powerful Elementor FAQ widget with accordion, search, categories, and full responsive controls.

## Features

- **Custom Post Type** - Manage FAQs through a dedicated WordPress admin interface
- **Categories Taxonomy** - Organize FAQs into categories for easy filtering
- **Accordion Mode** - Smooth open/close animations with configurable speed
- **Toggle Mode** - Choose between single (close others) or multiple open items
- **Real-time Search** - Filter FAQs instantly as users type
- **Category Filtering** - Horizontal or vertical tabs, plus mobile dropdown
- **Responsive Controls** - Adjust styles independently for desktop, tablet, and mobile
- **Elementor Native Styling** - Full control over typography, colors, borders, shadows
- **Custom Icons** - Set your own open/close icons using Elementor's icon picker
- **Animation Control** - Adjust animation speed to match your design

## Requirements

- WordPress 5.8 or higher
- PHP 7.4 or higher
- Elementor 3.0 or higher

## Installation

1. Download the plugin ZIP
2. Go to WordPress Admin > Plugins > Add New > Upload Plugin
3. Activate the plugin
4. Add FAQs via **FAQs > Add New**
5. Use the widget in Elementor (look in **Yosh Tools** category)

## Usage

### Creating FAQs

1. Navigate to **FAQs > Add New** in WordPress admin
2. Enter the question as the title
3. Enter the answer in the content editor
4. Assign categories (optional)
5. Set the order using Page Attributes
6. Publish

### Using the Widget

1. Edit a page with Elementor
2. Find the **FAQ** widget under **Yosh Tools** category
3. Drag and drop to your page
4. Configure source, behavior, and styling options

### Widget Settings

#### Content Tab
- **FAQ Source** - All FAQs, by Category, or Specific FAQs
- **Order By** - Menu order, Date, Title, or Random
- **Limit** - Maximum number of FAQs to display
- **Toggle Mode** - Single or Multiple open items
- **Enable Search** - Show/hide search input
- **Enable Categories** - Show/hide category filter tabs
- **Default Expand** - None, First FAQ, or All FAQs
- **Animation Speed** - Accordion animation duration in ms
- **Custom Icons** - Open/Close icons

#### Style Tab
- **Question** - Typography, colors (normal/hover/active), padding, border, icon styling
- **Answer** - Typography, colors, padding, border
- **Container** - Spacing, max width, box shadow, border
- **Category Tabs** - Typography, colors, padding, spacing, border radius
- **Search** - Typography, colors, padding, border
