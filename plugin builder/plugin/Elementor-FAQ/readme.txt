A powerful Elementor FAQ widget with accordion, search, categories, and full responsive controls.

== Description ==

Elementor FAQ is a powerful widget plugin that integrates seamlessly with Elementor Page Builder. Create beautiful FAQ sections with accordion functionality, real-time search, category filtering, and complete responsive design controls.

= Features =

* **Custom Post Type** - Manage FAQs through a dedicated admin interface
* **Categories** - Organize FAQs into categories for easy filtering
* **Accordion Mode** - Smooth open/close animations
* **Toggle Mode** - Choose between single or multiple open items
* **Real-time Search** - Filter FAQs instantly as users type
* **Category Filtering** - Horizontal or vertical tabs, plus mobile dropdown
* **Responsive Controls** - Adjust styles for desktop, tablet, and mobile
* **Elementor Native Styling** - Full control over typography, colors, borders, and more
* **Custom Icons** - Set your own open/close icons
* **Animation Control** - Adjust animation speed to match your design

= Requirements =

* WordPress 5.8 or higher
* PHP 7.4 or higher
* Elementor 3.0 or higher

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/elementor-faq` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to FAQs > Add New to create your FAQ items.
4. Use the Elementor editor to add the FAQ widget to any page.
5. Configure the widget settings and styles to match your design.

== Frequently Asked Questions ==

= How do I add FAQs? =

Go to FAQs > Add New in your WordPress admin. The title becomes the question, and the content editor is for the answer.

= Can I organize FAQs into categories? =

Yes! The plugin includes a FAQ Categories taxonomy. Assign categories to your FAQs and enable category filtering in the widget.

= Is the widget responsive? =

Absolutely! The widget includes full responsive controls for all styling options, plus automatic mobile optimization for category tabs.

= Does this work with the Elementor editor? =

Yes, this is a native Elementor widget. You'll find it under the "Yosh Tools" category in the Elementor widget panel.
